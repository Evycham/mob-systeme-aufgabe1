# mob-systeme1

Taschenrechner für Android.
Kurzes Android-Projekt (Java, Gradle Kotlin DSL) für die erste Aufgabe in Mobile Systeme.

## Voraussetzungen
- Android Studio (aktuelle Version)
- Android SDK `minSdk 33`, `targetSdk 36`

## Starten
1. Projekt in Android Studio öffnen.
2. Gradle-Sync abwarten.
3. App auf Emulator oder Gerät starten.

Author: Yurii Gruzevych

## Build per Terminal
```bash
./gradlew assembleDebug
```
